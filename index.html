Curso de Projeto Estrutural

Conteúdo do Curso

  • Cálculo de Esforços Simples - Normal, Cortante e Momento Fletor
    Normal: Refere-se aos esforços de compressão ou tração que atuam perpendicularmente à seção transversal de um elemento estrutural.
    Cortante: Relaciona-se aos esforços que agem tangencialmente à seção transversal de um elemento estrutural, causando corte na estrutura.
    Momento Fletor: Envolve os esforços que causam flexão em uma viga ou elemento estrutural, induzindo curvatura.
  • Pré-dimensionamento de Vigas, Lajes e Pilares
    Antes do cálculo detalhado, realiza-se uma estimativa inicial das dimensões destes elementos estruturais, considerando cargas e limites aceitáveis.
  • Cálculo e Detalhamento de Vigas
    Envolve a determinação das dimensões adequadas e do layout de armaduras para vigas, garantindo que elas suportem as cargas aplicadas.
  • Cálculo e Detalhamento de Lajes Maciças
    Similar às vigas, mas aplicado às lajes maciças, garantindo que a espessura e a distribuição da armadura sejam suficientes para suportar as cargas.
  • Cálculo e Detalhamento de Pilares
    Determina as dimensões e a armadura necessária para suportar as cargas verticais e garantir a estabilidade da estrutura.
  • Cálculo e Detalhamento de Fundação Rasa (Sapata Isolada)
    Envolve o projeto de fundações rasas, como sapatas isoladas, que distribuem as cargas das colunas para o solo.
  • Cálculo e Detalhamento de Fundação Profunda (Estaca - Compressão)
    Relacionado ao projeto de fundações profundas, como estacas, que são usadas quando o solo superficial não é adequado para suportar as cargas.
  • Cálculo de Elementos com Esforço Predominante a Flexão
    Aborda o dimensionamento de elementos estruturais onde os esforços de flexão são dominantes, como vigas e lajes em certas situações.
  • Introdução ao Software Eberick
    Apresenta o software Eberick, uma ferramenta de engenharia estrutural utilizada para análise e dimensionamento de estruturas.
  • Preparação da Arquitetura para Importação no Eberick
    Envolve a adaptação e configuração do modelo arquitetônico para ser importado no software Eberick.
  • Lançamento da Estrutura no Eberick
    Processo de entrada de dados e geometria no software Eberick para realizar análise e dimensionamento estrutural.
  • Aplicação dos Carregamentos Conforme as Normas Vigentes
    Considera a aplicação de cargas na estrutura de acordo com normas técnicas para garantir que o projeto atenda a requisitos de segurança.
  • Processamento, Análise e Eficiência do Projeto Estrutural
    Engloba a execução da análise estrutural no software Eberick, processamento de dados e avaliação da eficiência do projeto.
  • Detalhamento dos Elementos Estruturais e Formas
    Inclui a definição precisa das dimensões, posições e tipos de todos os elementos estruturais, além do detalhamento de suas formas construtivas.
